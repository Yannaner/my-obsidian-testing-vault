# VaultSite Plugin

An Obsidian plugin for publishing your vault as a beautiful website.

## Overview

VaultSite bridges the gap between Obsidian note-taking and web publishing, allowing you to:

- Export your notes to a Next.js-powered website
- Maintain folder structure and organization
- Support markdown features and syntax highlighting
- Deploy to Vercel with one click

## Features

### Markdown Support
- GitHub-flavored markdown
- Code syntax highlighting
- Tables and lists
- Blockquotes and links

### Customization
- Dark mode support
- Responsive design
- Custom themes (coming soon)

## Installation

1. Open Obsidian Settings
2. Go to Community Plugins
3. Search for "VaultSite"
4. Click Install
5. Enable the plugin

## Usage

After installation:

1. Run command: "VaultSite: Generate Site"
2. Configure your deployment settings
3. Deploy to Vercel or your preferred host
